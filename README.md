# JR-Player
基于AVPlayer的视频播放器。支持小屏、全屏播放。手势调节亮度、音量、播放速度等。支持视频缩略图预览。
播放器demo版本。仅实现基本播放功能。
##一、功能
  1. 视频全屏播放，根据重力感应自动调节屏幕方向。
  2. 播放器手势调节亮度、音量、播放进度。
  3. 添加视频内容预览功能。
  4. 进度条添加视频节点选取功能。可选取各个时间段视频截图做预览。
